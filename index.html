WatchSeries Downloader

WatchSeries Downloader is a tool which can be used to download videos from your anime or tv series automatically u just need to add it to your cron

Features

  • Its threaded on the background mode so it scans for 4 tv series at a time
  • Its can also be used in theminal mode using the postfix "--main""
  • You can give the starting episold from which u want it to download
  • Its completly in python
  • Dependes on wget

Usage

step 1: git clone the repo

git clone https://github.com/manojprithvee/Watchseries-Downloader.git

step 2: run the command for terminal based download and give a http://thewatchseries.to/episode/ or http://thewatchseries.to/serie/ link

python watchseries_downloader.py --main

or u can add all your links into the program and run it in the background using cron

python watchseries_downloader.py

to download from the backward 

python watchseries_downloader.py --reverse
